### 2020.06.20 22:50 周六(1st trail)

### 反馈建议
> 酱料应少放些  
> 温度应再高一些，烤得焦一些

### 耗时: 3h
准备：30min  
腌制：一夜(建议至少1h+)   
烘烤：30min  

### 材料
1. 鸡翅(10)
2. 老抽
3. 生抽
4. 料酒
5. 蚝油
6. 盐
7. 生姜
8. 蒜蓉
9. 孜然粉
10. 黑胡椒粉
11. 蜂蜜

### 步骤
1. 鸡翅洗干净(去血水)
2. 两面改刀(划3道口子,以碰到骨头即可)
3. ***完全凭感觉***倒入`材料2~10`
4. 戴上手套按摩抓均匀
5. 保鲜膜封上腌制(可放冰箱一夜；急了的话最少1h，一般2h)
6. 烤箱预热，垫上<a>锡纸</a>，刷上<span style='color:orange'>蜂蜜</span>（也可给鸡翅下面垫上土豆片）`中层上下火180°`烤`20min`，中间最好翻一次
7. 出锅！
