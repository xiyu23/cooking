## 烤鸡翅

### 次数
6+

最近一次：2023.1.29

### 材料（一次买一盒，盒马日日鲜360g的，一般9-12个，￥25）
1. 新鲜鸡翅6-7个（给媳妇吃孜然口味，剩下的我用奥尔良腌料弄）
2. 老抽一点点上色用
3. 生抽1.5陶瓷勺
4. 料酒0.5陶瓷勺
5. 蚝油0.5陶瓷勺
6. 盐一点点（大约1/4盐勺）
7. 孜然粉一点点
8. 孜然粒适量（需碾碎更容易出味儿）
9. 绵白糖0.5盐勺
10. 辣椒面适量
11. 蜂蜜0.5塑料勺

### 步骤
1. 鸡翅洗干净(去血水)
2. 两面改刀(划3道口子,以碰到骨头即可)，厨房用纸擦干水分，放入保鲜袋
3. 倒入材料`2~11`，挤弄均匀，放入冰箱，一般是腌一晚上，不过最少2h，试过5h也不错
4. 烤箱200度预热5min，烤盘垫上锡纸，纸上刷点儿植物油防粘，`中上层200°上下火`烤`20min`，中间最好翻一次（或空气炸锅200°，30min，中途翻面）
