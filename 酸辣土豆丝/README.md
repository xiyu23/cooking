

## 酸辣土豆丝

<!-- ![1](./pics/a.jpg) -->

### 耗时: 20min

### 时间
第3次 2022.08.03 周三 2min42s长了，太软了  
第2次 2022.07.29 周五 不错  
第1次 2022.07.28 周四 太辣了，少醋

### 材料
1. 土豆3个
2. 干辣椒4根
3. 线椒2根
4. 红色辣椒2根
5. 小葱2根
6. 白醋1勺
7. 盐0.75勺
8. 酱油0.75勺
9. 鸡精0.5勺

### 步骤
1. 热锅冷油，大火煸香干辣椒、绿辣椒、红辣椒（大约30s）
2. 倒入土豆丝翻炒
3. 加盐、白醋、酱油
4. 翻炒熟，加葱、鸡精
5. 出锅

### 关键点
1、全程大火，土豆丝入锅后2min~2min30s出锅合适
